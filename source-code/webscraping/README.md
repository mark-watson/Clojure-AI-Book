# Web Scraping — Example for "Practical Artificial Intelligence Programming With Clojure"

> **Book Chapter:** [Web Scraping](https://leanpub.com/read/clojureai/leanpub-auto-web-scraping) — free to read online.

This example demonstrates web scraping in Clojure using the Java [jsoup](https://jsoup.org/) library via interop. jsoup parses HTML into a DOM that you can query with CSS selectors — making it straightforward to extract text, links, and structured data from web pages.

## Prerequisites

| Tool | Version |
|------|---------|
| Java | 11+ |
| [Leiningen](https://leiningen.org) | 2.9+ |

No API keys are required.

## Run

Run the tests:

    lein test

Run the main demo:

    lein run
